# Day 10 Practice Assessment

This repository contains the solutions for the Day 10 Practice Assessment as part of the Wipro NGA .NET training program.

## Assignment 1 - Middleware Demo Application

In this assignment, I created an ASP.NET Core web application to understand how middleware works in the request-response pipeline.

### Features Implemented

* Request and response logging using custom middleware
* Error handling middleware
* Static file handling using the wwwroot folder
* HTTPS redirection
* Content Security Policy (CSP)
* HTML, CSS and JavaScript integration

### Technologies Used

* ASP.NET Core
* C#
* .NET

## Assignment 2 - Razor Pages Application

In this assignment, I developed a Razor Pages application that allows users to view and add items dynamically.

### Features Implemented

* Razor Pages architecture
* PageModel classes
* Razor syntax using foreach loop
* Data binding with BindProperty
* Form submission handling
* Dynamic item display

### Technologies Used

* ASP.NET Core Razor Pages
* C#
* .NET

* Assignment1_Middleware
* Assignment2_RazorItemsApp

Each project is organized according to standard ASP.NET Core project structure and naming conventions.

## How to Run the Projects

### Assignment 1

Open the Middleware project and run:

dotnet run

### Assignment 2

Open the Razor Pages project and run:

dotnet run

Then navigate to the appropriate Razor Pages URL to test the application.
